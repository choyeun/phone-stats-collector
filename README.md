# BGMonitor — Android 기기 상태 모니터링 앱

간단한 안드로이드 앱으로, 현재 폰의 상태를 한눈에 확인할 수 있습니다.

## 수집 정보

| 항목 | 상세 |
|---|---|
| 🔋 배터리 | 잔량, 온도, 전압, 충전상태, 건강도 |
| 💻 CPU / RAM | CPU 사용률, 메모리 사용량, 저장공간 |
| 🌐 네트워크 | 연결 타입 (LTE/5G/WiFi), 신호 세기 |
| 🔄 백그라운드 | 실행 중인 서비스 목록 |
| 📊 앱 사용량 | 최근 1시간 사용 앱 (UsageStats) |

## 사용 방법

1. 우측 Release에서 최신 APK 다운로드
2. 설치 후 실행
3. **사용량 접근 권한** 필요 시 설정 → 사용량 접근 → BGMonitor 활성화
4. 새로고침 버튼으로 정보 갱신

## 업데이트

앱 내 "업데이트 확인" 버튼으로 GitHub Release 자동 감지 및 설치 지원.
